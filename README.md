# DailyBoost Bot v10.0 — Голосовой AI-коуч

Запускается бесплатно на Render.com за 3 минуты  
Поддержка: русский, английский, испанский  
Голосовые сообщения + ежедневные напоминания

### Запуск:
1. Создай бота у @BotFather → получи токен
2. Залей этот код на GitHub
3. Зайди на https://render.com → New Web Service → Connect GitHub repo
4. Добавь переменную TELEGRAM_TOKEN
5. Жми "Create" → через 2 минуты бот живой!
